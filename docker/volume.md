# **`Managing Docker Volumes`**
---

## List Volumes
```bash
docker volume ls
```
List volumes.

## Create a Volume
```bash
docker volume create [OPTIONS] [VOLUME]
```
Create a volume.

## Inspect Volume
```bash
docker volume inspect [VOLUME...]
```
Display detailed information on one or more volumes.

## Remove a Volume
```bash
docker volume rm VOLUME [VOLUME...]
```
Remove one or more volumes.
